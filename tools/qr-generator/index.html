Frequently Asked Questions

What is a QR code? ▼

A QR (Quick Response) code is a 2D barcode that encodes text, URLs, or other data. Smartphones can scan them instantly with the camera app — no special reader needed.

What is error correction level? ▼

Error correction allows a QR code to be read even if part of it is damaged or covered. L=7%, M=15%, Q=25%, H=30% of the code can be obscured and still scan. Higher levels produce a denser pattern — use H if you plan to add a logo overlay.

Can I use custom colors? ▼

Yes. Pick any foreground and background color. Make sure there is strong contrast between the two — light on dark or dark on light — for scanners to reliably detect the pattern.

Is my data safe? ▼

Completely. The QR code is generated entirely inside your browser using JavaScript. No data is sent to any server.
